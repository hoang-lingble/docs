**Hero Banner Heading**

- As-is: Font size are currently static, not adjusting smoothly when the screen size changes.
- Suggestion: Fluid (40px Desktop / 28px Tablet / 22px Mobile).\
  OR Match current domestic site design.

White text + `rgba(0,0,0,0.5)` shadow for pop.

**Product Category Grid**

- As-is: Text is likely placed directly over denim image without sufficient separation, which would cause readability issue.
- Suggestion: White text + rgba(0,0,0,0.5) shadow for pop. With a proper shadow, customers can distinguish the message from the texture of the fabric in the background image.
